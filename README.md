# IP Logger

A simple and efficient IP logger built with Flask that tracks geolocation, device information, and visit statistics.

![Python](https://img.shields.io/badge/Python-3.7+-blue.svg)
![Flask](https://img.shields.io/badge/Flask-2.0+-green.svg)

## About the Project

IP Logger is a web application that automatically collects visitor information including:
- IP addresses
- Geolocation (country, city, coordinates)
- Device data (OS, browser, device type)
- Network information (ISP, ASN)
- Visit timestamps and visit counts
