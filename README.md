# IP Logger

Простой и эффективный IP логгер на Flask с отслеживанием геолокации, информации об устройстве и статистикой посещений.

![Python](https://img.shields.io/badge/Python-3.7+-blue.svg)
![Flask](https://img.shields.io/badge/Flask-2.0+-green.svg)

## О проекте

IP Logger - это веб-приложение, которое автоматически собирает информацию о посетителях сайта, включая:
- IP-адреса
- Геолокацию (страна, город, координаты)
- Данные об устройстве (ОС, браузер, тип устройства)
- Информацию о сети (провайдер, ASN)
- Время и количество посещений
